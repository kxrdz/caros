This directory contains the latest version of qhull to be linked with FreeSOLID

The distribution has been stripped of the documentation files and fitted with
autotools support in order to be linked seamlessly into FreeSOLID, 
Copyright notices and general information files have been maintained as 
requested by it's original authors.

For more info on Qhull, and to use it in your personal projects visit it's
webpage at http://www.qhull.org/.
